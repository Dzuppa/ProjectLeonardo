Confronto tra librerie grafiche per applicativi WPF:

MATERIAL DESIGN XAML

	PRO
	
	 - Molte combinazioni di colori tra cui scegliere
	 - Grande quantità di icone fornite dalla libreria
	 - Elemnti di UI che aggiunge ben fatti (le card sono davvero belle)
	 - Integrabile con MahApps
	 - Codice accessibile liberamente tramite GitHub
	 - Design ispirato a Google
	 - Completamente Open Source
	
	CONTRO
	
	 - Aggiunge pochi Elementi di UI (Card, orologi e progress bar)
	 - Documentazione inesistente e codice privo commenti 
	 - Design ispirato a Google
	 - Si concentra soprattutto sullo stilizzare elementi di UI già forniti di base
	 
	 

MAHAPPS 
	
	PRO
	
	 - Documentazione comprensiva di esempi sui control che aggiunge
	 - Aggiunge un numero leggermente maggiore di elementi di UI rispetto a MaterialDesign
	 - Integrabile con MaterialDesign
	 - Codice accessibile liberamente tramite GitHub
	 - Completamente Open Source
	 - Grande quantità di icone fornite dalla libreria
	 - Finestra dell'applicazione personalizzabile 
	 - Possibilità di aprire sottofinestre personalizzate
	
	
	CONTRO
	
	 - Stilizzazione degli elmenti di UI meno rilevante rispetto a MaterialDesign (si può però rimediare integrando le due)
	 - La documentazione non è presente per tutti i control ed, a volte, è poco accurata
	 - Non tutto quello che aggiunge è fatto con la stessa qualità
	 - Si concentra soprattutto sullo stilizzare elementi di UI già forniti di base
	 
	 
METROLIB 

	PRO
	
	 - Possibilità di fare grafici (ma sembrerebbero buggati)
	 - Codice liberamente accessibile tramite GitHub
	 - Licenza MIT
	
	CONTRO
	
	 - Assenza di documentazione ufficiale
	 - Nessuna icona fornita
	 - Graficamente meno interessante
	 - Elementi di UI aggiunti poco significativi
	 - Non aggiunge stile a molti elementi di UI forniti di base da Microsoft
	 - In generale alternativa decisamente meno interessante
	 - Scritta in .NET Framework 4.5 (non so se crea problemi di compatibilità ma alcuni control si vedono diversamente dalla demo della libreria)
	 - E' seguita da una sola persona, risulta quindi essere un progetto personale
	
	
MUI (Modern UI)

	PRO
	
	 - Codice liberamente accessibile tramite GitHub
	 - Navigazione tra pagine molto semplice ed efficace
	 - Layout delle pagine intuitivi da usare e belli da vedere
	
	CONTRO
	
	 - Scarsa documentazione ufficiale
	 - Non ci sono nuovi commit dal 22 Maggio 2016 (l'ultima versione ufficiale risale al 15 Marzo 2016)
	 - Stilizzazione degli elmenti di UI meno rilevante rispetto a MahApps 
	 - Fornisce poche icone e non sono applicabili e tutti i tipi di elementi di UI
	 - Graficamente meno interessante rispetto a MaterialDesign o MahApps
